## Pré-requisitos
- Apache Maven 3.9.9
- Java 21.0.5

## Instruções de instalação
1. Clone o repositório.
2. Entre no terminal e navegue até o diretório da aplicação
3. Execute ```mvn clean install```
4. Execute ```java -jar target/sistematizacao-0.0.1-SNAPSHOT.jar```
